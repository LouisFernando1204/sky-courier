## ğŸš€ Getting Started

### Prerequisites

- [GDevelop 5](https://gdevelop.io/) (version 5.5 or higher)

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/LouisFernando1204/sky-courier.git
   ```
2. Open GDevelop
3. Click "Open a Project"
4. Navigate to the project folder and select `Platformer.json`
5. Press the "Play" button to test the game

### Building for Web

1. In GDevelop, go to "File" > "Export"
2. Choose "Web (upload online or self-host)"
3. Follow the export wizard
4. Deploy to your preferred hosting service
